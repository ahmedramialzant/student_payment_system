نظام إدارة الرسوم الدراسية والأقساط

نظام رقمي ومحاسبي متكامل مصمم خصيصاً لأتمتة العمليات المالية وتتبع مستحقات الطلاب داخل المؤسسات التعليمية الخاصة والمراكز التدريبية، مع دعم كامل للتعامل بالعملة المحلية (الشيكل).

الميزات البرمجية والهندسية بالنظام:

الجدولة التلقائية للأقساط: احتساب قيمة القسط وتوليد تواريخ استحقاق شهرية منتظمة برمجياً فور تسجيل الطالب الجديد.

كفاءة بنية البيانات (Database Views): الاعتماد على استعلامات ذكية ومخزنة مثل v_student_balance لتخفيف الحمل على السيرفر وحصر الأرصدة لحظياً.

معالجة البيانات الفورية (AJAX): جلب وتحديث دفعات وأقساط الطلاب بالخلفية بشكل صامت وسريع دون الحاجة لإعادة تحميل كامل الصفحة.

الحماية والأمان العالي: تأمين مدخلات النظام بالكامل ضد ثغرات حقن الكود SQL Injection بالاعتماد على الاستعلامات المجهزة Prepared Statements (مكتبة PDO).

سجل الرقابة والتدقيق (Audit Log): نظام تعقب مركزي يوثق حركات الإدخال والتعديل والحذف برصد اسم المستخدم والوقت بالثانية مع التقاط عنوان الـ IP الحقيقي للجهاز المنفذ لضمان الشفافية.

تقنيات التطوير وبيئة التشغيل:

Back-End: PHP 8.2 (PDO Object)

Database: MySQL (InnoDB Engine)

Front-End: HTML5, CSS3, JavaScript (AJAX)

حزمة الخادم المحلي: XAMPP Server
